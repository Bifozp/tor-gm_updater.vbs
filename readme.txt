================================================================================
== Among Us Mod Updater for TheOtherRoles-GM (Steam)
================================================================================

[ 説明 ]
  Among Us のModのひとつ、「The Other Roles GM Edition」のインストール・および
  アップデートを自動化するスクリプトです。


[ 動作要件 ]
  ○使用している環境がWindows PCであること
    ・AndroidやiPhoneはもちろんのこと、MacOSやLinux等でも使用できません。

  ○Steamを使ってAmong Usをインストールしていること
    ・Epic Games等、Steam以外からインストールしている場合、
      このスクリプトは使えません。


[ 使用方法 ]
  ○Among Us のメジャーアップデートが行われている場合
    ・「aumod-cleaner.vbs」を起動し、Mod環境を一度クリーンアップして、
      Modの導入されていない（Vanilla）状態にする必要があります

  ○その他の場合
    ・「tor-gm_updater.vbs」を起動します。
      Modのダウンロードから所定の場所への解凍・配置までを自動でおこないます。


[ 免責 ]
  ・このスクリプトを使用して発生したいかなる問題についても、
    責任は負いかねます。


[ その他細かいこと ]
  ・スクリプトは通常、実行時に管理者権限での実行を要求されます。
    管理者権限で実行したくない場合は、vbsファイルをメモ帳などの
    テキストエディタで開き、４行目を以下の１行に差し替えてください。

    「Const enableUacCtl    = False」

    ※ "「" と "」" は 入力しないでください。
        ~~      ~~     ~~~~~~~~~~~~~~~~~~~~
    ※ aumod-cleaner.vbs も、 tor-gm_updater.vbs も手順は同じです。
